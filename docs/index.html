The loop

Role packs for agents exist. They're all stateless.

Week five's agent doesn't know what week two's learned. CB's whole bet is a weekly cycle where every role reads the rulebook before acting and writes back after — so the same team gets sharper every week it runs.

The rails

Non-negotiables, built into every role.

Outbox discipline

No tweet, email, or DM leaves without the human. Drafts land in outbox/ and stop there.

Read-first contract

Every role reads the dashboard, the strategy, and its own learnings file before acting.

Write-back contract

No run ends without updating the dashboard and logging what was durably learned.

Verdicts, not vibes

Gates close with READY / NEEDS WORK / NOT READY or SHIP / FIX — never narrative.

Humans decide

CB recommends. Cross-model or cross-role agreement is a signal, never permission.

Levers are typed

Every decision is registered as LEVER, JUDGMENT, or MECHANICAL. The test is blast radius, not file reversibility — and /chief audits for levers taken silently.
