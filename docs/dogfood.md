# Dogfood log — CB improving itself through real use

Convention (the continuous meta-reflection loop): every CB skill run inside a real project appends an entry here — what the skill was asked to do, where its text helped, where it chafed (missing mode, ambiguous step, absent output contract), and what fix was applied. Fixes follow /new-role discipline: match the form to the failure, fix only observed seams, one skill at a time. This log is also the raw material for the pre-v1.0 compounding demo.

## Entries

*(appended per run, newest first)*

- 2026-07-04 · **OWNER-CORRECTED (the most important entry so far)** · The agent's decision heuristic ("reversible + internal = agent decides") let two levers through silently: a positioning recommendation that became a narrative default, and a 15-context product taxonomy (an ontology commitment).
